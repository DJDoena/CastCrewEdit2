The folder "CastCrewEdit2" contains the headshots (if available) of cast and crew. 
The files are named by the IMDb code of the person.

The folder "DVD Profiler" contains the headshots (if available) of cast and crew. 
The files are named by the DVD Profiler naming convention.
The folder is emptied when you press "Get Headshots" and then filled with the headshots of the current IMDb title.
